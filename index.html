Proposta Comercial

Portal Imobiliário Royal Prime

Desenvolvimento de portal integrado à plataforma Imoview
Data: 06/04/2026
Validade: 10 dias corridos

1. Objeto

Desenvolvimento de um portal imobiliário web responsivo, com implementação completa do frontend a partir de layout fornecido pelo cliente, contemplando páginas de busca, listagem, detalhe, contato, anúncios, conteúdos institucionais e páginas legais, com integração à API do Imoview para consumo de dados e envio de formulários.

2. Escopo do Projeto

  • Frontend responsivo baseado em layout Figma ou PDF fornecido pelo cliente.
  • Aplicação em Next.js com SSR
  • Integração com API do Imoview para busca, listagem, detalhes, filtros e formulários.
  • Páginas de erro e tratamento visual para indisponibilidade.
  • Publicação inicial em ambiente do cliente.

Páginas Principais

  • Home
  • Buscar Imóveis / Listagem
  • Detalhes do Imóvel
  • Página de Locação
  • Página de Vendas

Páginas Institucionais

  • Contato
  • Anuncie seu Imóvel
  • Sobre
  • Documentos
  • LGPD

3. Tecnologias

Stack

  • Next.js
  • SSR
  • Rotas server/BFF

Integrações

  • API Imoview
  • Sem backend dedicado
  • Sem banco de dados próprio

4. Premissas

  • Dados provenientes exclusivamente da API do Imoview.
  • Formulários enviados para endpoints do Imoview.
  • Cliente fornece layout, credenciais, conteúdos e hospedagem.
  • Sem painel administrativo ou banco de dados próprio.
  • Integrações externas (WhatsApp, mapas, vídeos) de caráter simples.

5. Não Incluso

  • Criação de layout, UX ou identidade visual.
  • Contratação ou custeio de hospedagem.
  • Painel administrativo próprio.
  • Banco de dados próprio.
  • Produção de conteúdo textual.
  • Manutenção evolutiva fora da garantia.
  • Alterações por mudanças na API do Imoview.
  • Novas funcionalidades não descritas nesta proposta.

6. Dependências do Cliente

  • Layout final aprovado.
  • Documentação e credenciais da API do Imoview.
  • Ambiente de hospedagem.
  • Conteúdos institucionais e legais.
  • Validações nos marcos do projeto.

Nota: O acesso à documentação e credenciais da API é dependência crítica. Limitações ou restrições não documentadas da API do Imoview poderão implicar reavaliação de prazo e escopo.

7. Garantia

15 dias corridos após entrega final, limitada à correção de bugs e falhas do escopo contratado.

Não abrange: novas funcionalidades, alterações de layout pós-aprovação, mudanças de regra de negócio ou problemas de terceiros.

8. Suporte Mensal

Após a garantia, suporte opcional para correções, ajustes pontuais e acompanhamento operacional.

"O suporte mensal constitui serviço opcional e será contratado à parte, mediante valor mensal previamente acordado, contemplando correções pontuais, pequenos ajustes e apoio técnico relacionado ao funcionamento do portal."

9. Prazo

10 semanas, contadas do recebimento de todos os insumos: layout, credenciais da API e conteúdos.

Prazo sujeito a revisão em caso de atraso no envio de materiais ou validações.
